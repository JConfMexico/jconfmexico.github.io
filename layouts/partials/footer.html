About Us

JConf Mexico is a Java conference organized with love by two passionate Java User Groups: GDL JUG from Guadalajara and J4Guanatos from Guanajuato. Born from the desire to bring quality Java education and networking to the Mexican developer community, our event is created by developers for developers.
